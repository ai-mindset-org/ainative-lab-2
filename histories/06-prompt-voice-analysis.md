# Анализ голосовых заметок

---

## Почему голос

- Клавиатура: ~40 слов/мин
- Голос: ~150 слов/мин
- В голосовых мы не редактируем себя — говорим как думаем

---

## Что собирать

- Voice Memos (iOS)
- Telegram voice messages
- Транскрипты звонков (Granola, Krisp, Otter.ai)

---

## Если нет голосовых — наговори сейчас

```
Включи диктофон, говори 5-10 минут:

"Что было важного в моем 2025?
Какие темы повторялись?
Что радовало? Что беспокоило?
О чем думал в январе vs сейчас?"

Не структурируй, просто говори.
```

Затем транскрибируй через SuperWhisper, WisprFlow или MacWhisper.

---

## Промпт анализа

```
Вот транскрипты моих голосовых заметок за 2025:

[вставь текст]

Проанализируй:

1. Ключевые темы (5-7 топ по частоте)

2. Эмоциональные пики:
   - моменты радости/excitement
   - моменты тревоги/frustration

3. Recurring phrases — какие фразы повторяю?

4. State of mind по кварталам:
   - Q1: [какое настроение]
   - Q2: ...
   - Q3: ...
   - Q4: ...

5. Изменения фокуса: что планировал в начале года vs что обсуждаю сейчас?

Формат: markdown timeline с цитатами.
```

## Бонус: NVC промпт для SuperWhisper

Nonviolent Communication — переписывает сообщения без токсичности, сохраняя суть.

```
Твоя задача — переформулировать любой текст в стиль ненасильственного общения (NVC), не изменяя содержания, задачи, сроков и намерений.

Принципы:
- Убирай обвинительные формулировки
- Убирай императивный тон
- Сохраняй деловой стиль
- Не добавляй извинений и смягчений

Примеры:
"Сделай к вечеру" → "Нужно готово к вечеру."
"Почему это не готово?" → "Что сейчас мешает завершить?"
"Ты опять перепутал" → "Похоже, у нас разное понимание. Давай уточним."

Выводи только итоговый текст, без пояснений.
```
