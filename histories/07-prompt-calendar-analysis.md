# Анализ календаря

---

## Как экспортировать

**Google Calendar:**
Settings → Import & Export → Export → .ics файл

**Apple Calendar:**
File → Export → выбрать 2025 → .ics файл

**Альтернатива:** скриншоты по месяцам

---

## Промпт

```
Вот мой календарь за 2025 год.

[вставь .ics файл или summary по месяцам]

Проанализируй:

1. Пиковые месяцы (по количеству событий)
   - какой месяц был самым загруженным?
   - correlation с результатами/проектами?

2. "Ямы" без активности
   - периоды без встреч > 3 дней
   - что там было? отдых? burnout?

3. Топ-10 людей (по частоте встреч)
   - с кем проводил больше всего времени?
   - рабочие vs личные?

4. Категории событий:
   - work meetings (%)
   - personal time (%)
   - health/sports (%)
   - social events (%)

5. Регулярные паттерны:
   - еженедельные recurring events
   - ежемесячные ритуалы

Создай визуализацию:
     Jan  Feb  Mar  Apr  May  Jun  Jul  Aug  Sep  Oct  Nov  Dec
     [  ][  ][██][  ][  ][  ][  ][░░][  ][  ][██][  ]

+ insights: что календарь говорит о моих реальных приоритетах?
```

---

## Дополнительно: correlation с энергией

```
Сопоставь плотность календаря с моим состоянием:

Плотные месяцы: [список из анализа]
Спокойные месяцы: [список]

Мои воспоминания:
- когда чувствовал себя лучше всего?
- когда был burnout?
- когда была максимальная продуктивность?

Найди correlation: busy calendar → stress или busy → flow?
```
